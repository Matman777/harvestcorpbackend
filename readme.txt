J'ai installé express avec "npm install express"

Ensuite, je vais éxécuter "npm i sequelize mariadb" pour installer sequelize (ORM) et mariadb (driver pour connecter à la BDD)

Branche "login":
Installation de Bcrypt pour crypter et décrypter les mots de passe
Installation de Jsonwebtoken pour gérer la création d'un token et la gestion des sessions
Installation de Cors pour permettre à d'autres applications de faire des requêtes sur l'API
